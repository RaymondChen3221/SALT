# SALT v4 题目设计说明

## 结构

v4 一共 44 题：

- 主表 36 题：S / A / L 三个轴，每轴 Self 6 题、Other 6 题
- 支持偏好 8 题：实用托底、情绪理解、共同创作、低压在场各 2 题

每题都有：

```js
{
  id,
  section,
  axis or support,
  perspective: "self" | "other" | null,
  pair,
  direction,
  text
}
```

## 配对逻辑

Self/Other 仍然使用同一个 `pair` 做 T 差值计算，但 v4 不再把 Other 题写成 Self 题的镜像改写。每组题测量同一子维度，但换了场景或表达：

- S1：专属称呼、暗号、表达
- S2：在多人友好关系中的特殊位置
- S3：是否需要区分喜欢的人和普通朋友，反向题
- S4：吃醋或位置不安时是否给确认
- S5：两人内容被公开化后的影响，反向题
- S6：私密内容和边界
- A1：承诺小事失败时是否提前说明
- A2：在意是否落成行动
- A3：关系问题里消失或回避，反向题
- A4：受伤后的承认与修复
- A5：只停在心里是否也够，反向题
- A6：接不住时是否给最小信号
- L1：是否自然把对方放进未来设想
- L2：关系是否影响时间安排和生活结构
- L3：是否只看当下舒适，反向题
- L4：是否进入生活场景、兴趣和计划
- L5：是否因为关系调整生活结构，反向题
- L6：是否在意生日、约定、纪念日等节点

## 方向平衡

S / A / L 每个轴都包含正向和反向题。反向题不是简单否定同一句话，而是用另一种自然关系场景测量低特殊性、低兑现力或低长程性的倾向。

支持偏好题全部按原始答案计分，不参与 SALT 类型码。
